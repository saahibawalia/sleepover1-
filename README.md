Cute Animal Carousel Website

## Overview

This project is a simple webpage that shows different animals using polaroid-style cards. Each card contains an image of an animal and a button that links to a YouTube video related to that animal.

The cards move across the screen in a continuous carousel animation.

## Features

* Polaroid-style card layout
* Multiple animals (cats, dogs, bunnies, parrots)
* Buttons that open YouTube videos
* Automatic carousel animation using CSS
* Simple and clean design

## Technologies Used

* HTML
* CSS
* CSS animations

## How the Carousel Works

All of the cards are placed inside a moving track. The track moves to the left using a CSS animation. The cards are duplicated so that when the first set moves off the screen, the next set appears, creating a continuous loop.
